Requirements
.Net 4+
IE 8+


Paste the following in cmd to see the installed .Net versions
dir /b /ad /o-n %systemroot%\Microsoft.NET\Framework\v?.*

You'll see something like this
v4.0.30319 ---If it is not there then you should install .Net 4+
v3.5
v3.0
v2.0.50727
v1.1.4322
v1.0.3705

Open Register.bat for installation
Open Unregister.bat for uninstallation
